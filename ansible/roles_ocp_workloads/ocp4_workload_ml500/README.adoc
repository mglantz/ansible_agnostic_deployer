= ocp4-workload-ml500 - Deploy ML500 (AI500) infrastructure

== Role overview

* This role install link:https://github.com/rhoai-mlops/deploy-lab.git[ML500] course content into OpenShift
** Playbook: link:./tasks/workload.yml[workload.yml]

=== Deploy a Workload with the `ocp-workload` playbook

Create var file as below:

```yaml
---
cloud_provider: none
env_type: ocp-workloads

ocp_username: 
ocp_password: 
guid: guid

ocp_workloads:
  - ocp4_workload_ml500

target_host:
  ansible_host: bastion.guid.sandbox123.opentlc.com
  ansible_port: 22
  ansible_user: lab-user
  ansible_ssh_private_key_file: ~/.ssh/id_rsa
```

Then run the below commands:
```bash
cd ansible
ansible-playbook main.yml -e @<above-file>
```
