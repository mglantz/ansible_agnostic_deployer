Role: infra-ec2-open-ssh-access
===============================

This ensures that all security groups in the `aws_region` allow SSH access from
the IP address from which Ansible is executing.

Requirements
------------

* Common AWS variables are set:
** `aws_region`
** `aws_access_key_id`
** `aws_secret_access_key`
