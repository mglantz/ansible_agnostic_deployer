Role: collect_openshift_must_gather
===================================

This role runs an OpenShift `oc adm must-gather` to collect information from an OpenShift cluster.
Results are saved in the `output_dir`.

Requirements
------------

* The `oc` command must be available on localhost or execution environment.
* OpenShift credentials must be available:
** In local environment or via environment variables.
** In kubeconfig specified by `collect_openshift_must_gather_kubeconfig`
** In AgnosticD user data (see `defaults/main.yml`)
