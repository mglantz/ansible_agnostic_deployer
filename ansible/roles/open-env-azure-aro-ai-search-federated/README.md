Role Name
=========

This role creates an Azure federated identity credential linking the service account in OpenShift Container Platform to the Azure user-assigned managed identity.

Reference : https://github.com/rhpds/lb2768-building-security-focused-ai-applications-using-managed-identities-on-aro/blob/main/ARO_MIWI.md

Requirements
------------

az command in virtualenv

Role Variables
--------------

guid - the guid to use for the deployment
requester_email - an email address to invite
